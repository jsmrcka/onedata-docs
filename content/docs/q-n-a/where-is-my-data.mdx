---
title: "Where are my data stored?"
---

The Onedata filesystem works through a network of Providers (data centers) that sync data across shared spaces. 
Here is how it manages your files:
* __Smart Distribution__: Files are broken into blocks and distributed across different Providers as replicas.
* __Metadata Syncing__: Information about where these blocks are stored is automatically synchronized between all Providers.
* __Conflict-Free Editing__: Multiple users can edit a file at once. If changes don't overlap, they are all saved. If there’s a conflict, an automated algorithm ensures every Provider reaches the same final version quickly, without needing to wait for one another. 

<Callout type="info" title="Note" icon="📌">
  All Providers managed by e-INFRA CZ Onedata service (i.e. not external or user-managed Providers) are backed by data storage infrastructure
  operated by CESNET and located in the Czech Republic.

  For details, see:
  - https://docs.du.cesnet.cz/en/docs/introduction/description
</Callout>

You can discover how the file blocks are distributed among providers supporting the space in which it is stored via Web GUI:  
1. Open the context menu for the file and choose Data distribution.
